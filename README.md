# seed-kubernetes

## Installation
1. [Docker](https://docs.docker.com/compose/install/)

3. [Kubectl](https://kubernetes.io/docs/reference/kubectl/)

## Notes
* OS used: Linux.
* Maybe it's needed to run the installation and commands with sudo.

## Config file
